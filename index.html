Mock URL Format and Existence Validator

This is a mock URL Validator to mock the validation and checking of existence of the path or file

For rules of URL, see here

As the server side is not implemented, only URL with http and https schemes are considered as valid.

Besides the limitation above, only the domains fakedomain.com and fake-domain-http.com is allowed, and following URLs of folders/files be considered as existed:

  • https://fakedomain.com/fake-path
  • https://fakedomain.com/fake-path/fake-files
  • https://fakedomain.com/fake-path/fake-files/faketext.txt
  • http://fake-domain-http.com/fake-folder/faketext.txt
